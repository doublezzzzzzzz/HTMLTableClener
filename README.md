# HTML Table Cleaner & SQLite Exporter

**HTML Table Cleaner** — это профессиональное настольное приложение на **C++/Qt**, предназначенное для интеллектуальной очистки «мусорного» исходного кода HTML файла (экспортированного из MS Word) и преобразования таблиц в структурированную базу данных SQLite.

---

## Ключевые возможности

-  **Глубокая очистка**: Удаление инлайновых стилей, офисных метаданных и пустых тегов.
-  **Виртуальная сетка**: Уникальный алгоритм восстановления геометрии таблиц.
-  **Умные ячейки**: Архитектура `RealCell` / `ReferenceCell` на базе указателей для обработки `rowspan` и `colspan` без дублирования данных.
-  **Быстрый экспорт**: Транзакционная запись в SQLite с автоматическим созданием схем таблиц.
-  **Live Preview**: Мгновенная визуализация результата парсинга в приложении.

---

##  Технический стек

* **Язык**: C++17
* **Интерфейс**: Qt Widgets
* **БД**: QSql (SQLite)
* **Сборка**: CMake 3.16+

---

##  Архитектура проекта

Программа использует многоуровневую систему обработки данных:

1. **Pipeline очистки**: Последовательное применение фильтров к исходному коду HTML файла.
2. **Координатная модель**: Каждая ячейка получает свои координаты `(x, y)` в виртуальной матрице.
3. **Reference System**: Объединенные ячейки не копируют текст, а ссылаются на "родительскую" ячейку через `std::shared_ptr`.

| Клавиши | Действие |
| :--- | :--- |
| <kbd>Ctrl</kbd> + <kbd>O</kbd> | Открыть файл |

| <kbd>Ctrl</kbd> + <kbd>E</kbd> | Экспорт в SQLite |

| <kbd>Esc</kbd> | Очистить всё |

##  Установка и сборка

Для компиляции вам понадобятся **Qt 5.15+** и **CMake**.

```bash
# Клонирование репозитория
git clone [https://github.com/yourusername/html-table-cleaner.git](https://github.com/yourusername/html-table-cleaner.git)
cd html-table-cleaner

# Сборка проекта
mkdir build && cd build
cmake ..
cmake --build . --config Release
