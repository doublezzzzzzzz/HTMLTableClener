## Обзор проекта

**HTML Table Cleaner** — это настольное приложение на Qt, предназначенное для очистки и нормализации HTML-таблиц, особенно тех, которые экспортируются из Microsoft Word. Приложение преобразует "грязный" HTML-код с множеством стилей, классов и атрибутов в чистый, минималистичный HTML-код, сохраняя при этом структуру таблицы.

### Основные возможности:
- Извлечение таблиц из HTML-кода, экспортированного из Word
- Удаление Word-специфичных стилей, классов и атрибутов
- Удаление тегов форматирования с сохранением текстового содержимого
- Нормализация HTML-структуры
- Сохранение структуры таблицы (строки, ячейки, rowspan/colspan)
- Гибкий пользовательский интерфейс с поддержкой горячих клавиш

#### mainwindow.cpp

**Назначение:** Реализация всех методов класса `MainWindow`.

**Зависимости:**
- `mainwindow.h`
- `ui_mainwindow.h` (автоматически сгенерирован)
- `TableParser.h`
- Различные модули Qt (QFileDialog, QMessageBox, QFile, QTextStream, QClipboard, QApplication, QShortcut)

**Основные методы:**

**MainWindow::MainWindow(QWidget *parent)**
- **Назначение:** Конструктор главного окна
- **Действия:**
  - Инициализирует интерфейс через `ui->setupUi(this)`
  - Подключает сигналы кнопок к соответствующим слотам
  - Настраивает горячие клавиши для всех кнопок:
    - Открыть: `Ctrl+O`
    - Очистить HTML: `Ctrl+R`
    - Сохранить: `Ctrl+S`
    - Копировать: `Ctrl+C`
    - Очистить поля: `Ctrl+L` или `Esc`
  - Устанавливает начальное сообщение в статусной строке

**MainWindow::onOpenButtonClicked()**
- **Назначение:** Обработка открытия HTML-файла
- **Действия:**
  - Открывает диалог выбора файла (фильтр: `.html`, `.htm`)
  - Читает содержимое файла с кодировкой UTF-8
  - Загружает содержимое в текстовое поле исходного HTML
  - Обновляет статусную строку с именем загруженного файла
- **Обработка ошибок:** Выводит предупреждение, если файл не удалось открыть

**MainWindow::onCleanButtonClicked()**
- **Назначение:** Обработка очистки HTML-таблицы
- **Действия:**
  - Получает исходный HTML-код из текстового поля
  - Проверяет, что поле не пустое
  - Вызывает `TableParser::cleanWordHtml()` для очистки HTML
  - Парсит очищенный HTML в объект `Table`
  - Обновляет отображение результата через `updateResultDisplay()`
  - Обновляет статусную строку с информацией о количестве строк
- **Обработка ошибок:**
  - Предупреждение, если исходное поле пустое
  - Предупреждение, если таблица не найдена
  - Критическое сообщение при исключениях

**MainWindow::updateResultDisplay()**
- **Назначение:** Обновление текстового поля с очищенным HTML
- **Действия:**
  - Конвертирует объект `Table` в HTML-строку через `toHtml()`
  - Загружает результат в текстовое поле `resultTextEdit`

**MainWindow::onSaveButtonClicked()**
- **Назначение:** Сохранение очищенного HTML в файл
- **Действия:**
  - Проверяет наличие данных для сохранения
  - Открывает диалог сохранения файла
  - Автоматически добавляет расширение `.html`, если не указано
  - Записывает очищенный HTML в файл с кодировкой UTF-8
  - Обновляет статусную строку
- **Обработка ошибок:** Предупреждение при ошибках записи

**MainWindow::onCopyButtonClicked()**
- **Назначение:** Копирование очищенного HTML в буфер обмена
- **Действия:**
  - Проверяет наличие данных для копирования
  - Копирует текст из поля результата в системный буфер обмена
  - Обновляет статусную строку с информацией о количестве скопированных символов и строк
  - Сообщение отображается 3 секунды

**MainWindow::onClearButtonClicked()**
- **Назначение:** Очистка всех полей
- **Действия:**
  - Проверяет наличие данных в полях
  - Показывает диалог подтверждения (если есть данные)
  - Вызывает `clearAllFields()` для очистки

**MainWindow::clearAllFields()**
- **Назначение:** Очистка всех полей и данных
- **Действия:**
  - Очищает текстовое поле исходного HTML
  - Очищает текстовое поле результата
  - Очищает объект `currentTable`
  - Обновляет статусную строку

### Table.h и Table.cpp

#### Table.h

**Назначение:** Определяет класс `Table`, который представляет структуру таблицы.

**Зависимости:**
- `Cell.h`
- `<vector>`

**Класс Table:**

**Публичные методы:**
- `Table() = default` — конструктор по умолчанию
- `void addRow(const std::vector<Cell>& row)` — добавление строки в таблицу
- `const std::vector<std::vector<Cell>>& getRows() const` — получение всех строк таблицы (read-only)
- `void clear()` — очистка таблицы (удаление всех строк)
- `std::string toHtml() const` — преобразование таблицы в HTML-строку

**Приватные члены:**
- `std::vector<std::vector<Cell>> rows` — контейнер строк таблицы (каждая строка — вектор ячеек)

#### Table.cpp

**Назначение:** Реализация методов класса `Table`.

**Зависимости:**
- `Table.h`
- `<sstream>` (для `std::ostringstream`)

**Методы:**

**Table::addRow(const std::vector<Cell>& row)**
- **Назначение:** Добавление строки в таблицу
- **Параметры:** `row` — вектор ячеек, представляющий строку таблицы
- **Действия:** Добавляет строку в конец контейнера `rows`

**Table::toHtml() const**
- **Назначение:** Преобразование таблицы в HTML-строку
- **Возвращаемое значение:** `std::string` — HTML-код таблицы
- **Действия:**
  - Создает HTML-тег `<table border="1">`
  - Для каждой строки создает тег `<tr>`
  - Для каждой ячейки создает тег `<td>` или `<th>` (в зависимости от `cell.isHeader`)
  - Добавляет атрибуты `rowspan` и `colspan`, если они больше 1
  - Вставляет содержимое ячейки (`cell.content`)
  - Закрывает все теги
- **Формат выходного HTML:**
  ```html
  <table border="1">
    <tr>
      <td>содержимое</td>
      <th rowspan="2" colspan="3">заголовок</th>
    </tr>
    ...
  </table>
  ```

### Cell.h

**Назначение:** Определяет структуру `Cell`, представляющую ячейку таблицы.

**Зависимости:**
- `<string>`

**Структура Cell:**

**Поля:**
- `int rowspan = 1` — количество строк, которые занимает ячейка (по умолчанию 1)
- `int colspan = 1` — количество столбцов, которые занимает ячейка (по умолчанию 1)
- `std::string content` — текстовое содержимое ячейки
- `bool isHeader = false` — флаг, указывающий, является ли ячейка заголовком (`<th>`) или обычной ячейкой (`<td>`)

**Конструкторы:**
- `Cell() = default` — конструктор по умолчанию (инициализирует поля значениями по умолчанию)
- `Cell(const std::string& content, int rowspan = 1, int colspan = 1, bool isHeader = false)` — конструктор с параметрами

**Использование:**
Структура используется для представления ячеек таблицы в объектах класса `Table`. Позволяет хранить не только содержимое, но и атрибуты форматирования (rowspan, colspan) и тип ячейки (заголовок или обычная).

### TableParser.h и TableParser.cpp

#### TableParser.h

**Назначение:** Определяет класс `TableParser`, содержащий статические методы для парсинга и очистки HTML-таблиц.

**Зависимости:**
- `Table.h`
- `<string>`

**Класс TableParser:**

**Особенность:** Все методы класса статические, экземпляр класса не создается.

**Публичные статические методы:**
- `static Table parseSimpleHtml(const std::string& html)` — парсинг простого HTML-кода таблицы
- `static std::string extractTableFromWordHtml(const std::string& html)` — извлечение таблицы из Word HTML
- `static std::string cleanWordHtml(const std::string& html)` — комплексная очистка Word HTML

**Приватные статические методы:**
- `static std::string cleanCellContent(const std::string& content)` — очистка содержимого ячейки
- `static int extractRowspan(const std::string& html, size_t pos)` — извлечение атрибута rowspan
- `static int extractColspan(const std::string& html, size_t pos)` — извлечение атрибута colspan
- `static std::string removeWordStyles(const std::string& html)` — удаление Word-специфичных стилей
- `static std::string normalizeHtml(const std::string& html)` — нормализация HTML-кода

#### TableParser.cpp

**Назначение:** Реализация всех методов класса `TableParser` и вспомогательных функций.

**Зависимости:**
- `TableParser.h`
- `<iostream>` (для отладочного вывода)
- `<algorithm>` (для `transform`, `tolower`)
- `<cctype>` (для `isspace`)

**Вспомогательные функции (не члены класса):**

**findMatchingCloseTag(const string& html, const string& tag, size_t start_pos)**
- **Назначение:** Поиск соответствующего закрывающего тега с учетом вложенности
- **Параметры:**
  - `html` — HTML-строка для поиска
  - `tag` — имя тега (без угловых скобок)
  - `start_pos` — позиция начала поиска
- **Возвращаемое значение:** `size_t` — позиция закрывающего тега или `string::npos`
- **Алгоритм:** Отслеживает уровень вложенности тегов для корректного поиска закрывающего тега

**extractAttribute(const string& html, const string& attr_name, size_t pos)**
- **Назначение:** Извлечение значения атрибута из HTML-тега
- **Параметры:**
  - `html` — HTML-строка
  - `attr_name` — имя атрибута
  - `pos` — позиция начала поиска
- **Возвращаемое значение:** `string` — значение атрибута или "1" по умолчанию
- **Особенности:** Обрабатывает пробелы, кавычки и табуляции вокруг знака равенства

**removeFormattingTags(const string& text)**
- **Назначение:** Удаление тегов форматирования из текста
- **Параметры:** `text` — текст с HTML-тегами
- **Возвращаемое значение:** `string` — текст без тегов форматирования
- **Обрабатываемые теги:**
  - Жирный: `<b>`, `<B>`, `<strong>`, `<STRONG>`
  - Курсив: `<i>`, `<I>`, `<em>`, `<EM>`
  - Подчеркивание: `<u>`, `<U>`, `<ins>`, `<INS>`
  - Зачеркивание: `<s>`, `<S>`, `<strike>`, `<STRIKE>`, `<del>`, `<DEL>`
  - Прочие: `<span>`, `<SPAN>`, `<font>`, `<FONT>`
- **Алгоритм:** Рекурсивно удаляет теги, сохраняя текстовое содержимое

**Методы класса TableParser:**

**TableParser::cleanCellContent(const std::string& content)**
- **Назначение:** Комплексная очистка содержимого ячейки таблицы
- **Параметры:** `content` — исходное содержимое ячейки
- **Возвращаемое значение:** `std::string` — очищенное содержимое
- **Этапы обработки:**
  1. Удаление тегов форматирования через `removeFormattingTags()`
  2. Обработка параграфов (`<p>...</p>`):
     - Сохранение структуры параграфов
     - Удаление атрибутов из тегов `<p>`
     - Удаление `<br>` тегов (замена на пробелы)
     - Нормализация пробелов
  3. Удаление HTML-сущностей (`&nbsp;`, `&amp;`, `&lt;`, `&gt;`, `&quot;`, `&apos;`, `&#39;`, `&#34;`, `&#160;`, `&ensp;`, `&emsp;`)
  4. Удаление пустых параграфов (`<p></p>`, `<p> </p>`)
  5. Нормализация множественных пробелов
  6. Обрезка пробелов по краям

**TableParser::removeWordStyles(const std::string& html)**
- **Назначение:** Удаление Word-специфичных стилей и атрибутов
- **Параметры:** `html` — HTML-код
- **Возвращаемое значение:** `std::string` — HTML без Word-стилей
- **Удаляемые элементы:**
  - Атрибуты `class="..."` (особенно MSO классы)
  - Атрибуты `style="..."`
  - XML namespace declarations (`xmlns:v=`, `xmlns:o=`, `xmlns:w=`, `xmlns:m=`, `v:lang=`, `o:lang=`, `lang=`, `xmlns=`)

**TableParser::normalizeHtml(const std::string& html)**
- **Назначение:** Нормализация HTML-кода (удаление ненужных элементов, сохранение структуры таблицы)
- **Параметры:** `html` — исходный HTML
- **Возвращаемое значение:** `std::string` — нормализованный HTML
- **Действия:**
  1. Удаление HTML-комментариев (`<!-- ... -->`)
  2. Удаление CDATA секций (`<![CDATA[...]]>`)
  3. Удаление всех тегов, кроме разрешенных:
     - `<table>`, `<tr>`, `<td>`, `<th>`, `<p>`, `<br>`
     - Их закрывающие теги
     - `<tbody>`, `</tbody>` (обрабатывается отдельно)
  4. Удаление пустых тегов `<tbody>...</tbody>` (замена на содержимое)

**TableParser::extractTableFromWordHtml(const std::string& html)**
- **Назначение:** Извлечение таблицы из полного HTML-кода Word-документа
- **Параметры:** `html` — полный HTML-код
- **Возвращаемое значение:** `std::string` — HTML-код только таблицы
- **Действия:**
  1. Нормализация HTML через `normalizeHtml()`
  2. Поиск первого тега `<table>`
  3. Поиск соответствующего закрывающего тега `</table>` через `findMatchingCloseTag()`
  4. Извлечение HTML-кода таблицы
  5. Удаление Word-стилей через `removeWordStyles()`
- **Отладочный вывод:** Размер HTML и извлеченной таблицы

**TableParser::extractRowspan(const std::string& html, size_t pos)**
- **Назначение:** Извлечение значения атрибута `rowspan` из HTML-тега
- **Параметры:**
  - `html` — HTML-строка
  - `pos` — позиция начала поиска
- **Возвращаемое значение:** `int` — значение rowspan или 1 по умолчанию
- **Использование:** Использует `extractAttribute()` и преобразует строку в число

**TableParser::extractColspan(const std::string& html, size_t pos)**
- **Назначение:** Извлечение значения атрибута `colspan` из HTML-тега
- **Параметры:**
  - `html` — HTML-строка
  - `pos` — позиция начала поиска
- **Возвращаемое значение:** `int` — значение colspan или 1 по умолчанию
- **Использование:** Аналогично `extractRowspan()`

**TableParser::parseSimpleHtml(const std::string& html)**
- **Назначение:** Парсинг простого HTML-кода таблицы в объект `Table`
- **Параметры:** `html` — HTML-код таблицы (уже очищенный)
- **Возвращаемое значение:** `Table` — объект таблицы
- **Алгоритм:**
  1. Проверка на пустой HTML
  2. Поиск всех строк (`<tr>...</tr>`):
     - Использование `findMatchingCloseTag()` для корректной обработки вложенности
     - Извлечение HTML-кода строки
  3. Для каждой строки поиск всех ячеек (`<td>...</td>` и `<th>...</th>`):
     - Определение типа ячейки (td или th)
     - Извлечение атрибутов `rowspan` и `colspan`
     - Извлечение содержимого ячейки
     - Очистка содержимого через `cleanCellContent()`
     - Создание объекта `Cell` и добавление в строку
  4. Добавление строки в таблицу через `table.addRow()`
- **Отладочный вывод:** Информация о процессе парсинга (количество строк, ячеек)

**TableParser::cleanWordHtml(const std::string& html)**
- **Назначение:** Комплексная очистка HTML-кода Word-документа и извлечение таблицы
- **Параметры:** `html` — полный HTML-код Word-документа
- **Возвращаемое значение:** `std::string` — чистый HTML-код таблицы
- **Алгоритм:**
  1. Извлечение таблицы через `extractTableFromWordHtml()`
  2. Если таблица найдена, парсинг через `parseSimpleHtml()`
  3. Преобразование обратно в HTML через `table.toHtml()`
- **Возвращает:** Пустую строку, если таблица не найдена

## Детальное описание классов и структур

### Структура Cell

**Назначение:** Представляет ячейку HTML-таблицы.

**Поля:**
| Поле | Тип | Значение по умолчанию | Описание |
|------|-----|----------------------|----------|
| `rowspan` | `int` | 1 | Количество строк, которые занимает ячейка |
| `colspan` | `int` | 1 | Количество столбцов, которые занимает ячейка |
| `content` | `std::string` | "" | Текстовое содержимое ячейки |
| `isHeader` | `bool` | false | Флаг заголовка (true = `<th>`, false = `<td>`) |

**Конструкторы:**
- `Cell()` — конструктор по умолчанию, инициализирует все поля значениями по умолчанию
- `Cell(const std::string& content, int rowspan = 1, int colspan = 1, bool isHeader = false)` — конструктор с параметрами

**Использование:**
Ячейки хранятся в векторах, которые представляют строки таблицы. Объекты `Cell` создаются в процессе парсинга HTML и используются для построения структуры таблицы в классе `Table`.

---

### Класс Table

**Назначение:** Представляет структуру HTML-таблицы как контейнер строк ячеек.

**Приватные члены:**
- `std::vector<std::vector<Cell>> rows` — двумерный контейнер: внешний вектор — строки, внутренний — ячейки в строке

**Методы:**

**addRow(const std::vector<Cell>& row)**
- Добавляет новую строку в конец таблицы
- **Параметры:** `row` — вектор ячеек для добавления
- **Исключения:** Нет

**getRows() const**
- Возвращает константную ссылку на все строки таблицы
- **Возвращаемое значение:** `const std::vector<std::vector<Cell>>&`
- **Использование:** Для чтения структуры таблицы без возможности модификации

**clear()**
- Удаляет все строки из таблицы
- **Использование:** При очистке данных в интерфейсе

**toHtml() const**
- Преобразует таблицу в HTML-код
- **Возвращаемое значение:** `std::string` — валидный HTML-код
- **Формат вывода:**
  - Открывающий тег: `<table border="1">`
  - Для каждой строки: `<tr>...</tr>`
  - Для каждой ячейки: `<td>...</td>` или `<th>...</th>`
  - Атрибуты `rowspan` и `colspan` добавляются только если > 1
  - Закрывающий тег: `</table>`
- **Отступы:** Используются для читаемости (2 пробела для строк, 4 для ячеек)

---

### Класс TableParser

**Назначение:** Утилитарный класс для парсинга и очистки HTML-таблиц. Все методы статические.

**Особенности:**
- Не создаются экземпляры класса (все методы статические)
- Основной класс для обработки HTML-кода
- Содержит как публичные (интерфейсные), так и приватные (вспомогательные) методы

**Публичные методы (интерфейс):**

1. **parseSimpleHtml(const std::string& html)**
   - Парсинг уже очищенного HTML-кода таблицы
   - Используется после предварительной очистки
   
2. **extractTableFromWordHtml(const std::string& html)**
   - Извлечение таблицы из полного HTML-кода Word-документа
   - Возвращает только HTML-код таблицы
   
3. **cleanWordHtml(const std::string& html)**
   - Главный метод для комплексной очистки
   - Объединяет извлечение и очистку таблицы
   - Возвращает готовый чистый HTML-код

**Приватные методы (вспомогательные):**

1. **cleanCellContent(const std::string& content)**
   - Самая сложная функция очистки
   - Обрабатывает форматирование, параграфы, HTML-сущности
   
2. **extractRowspan/extractColspan(const std::string& html, size_t pos)**
   - Извлечение атрибутов из HTML-тегов
   
3. **removeWordStyles(const std::string& html)**
   - Удаление Word-специфичных элементов
   
4. **normalizeHtml(const std::string& html)**
   - Базовая нормализация HTML-структуры

**Вспомогательные функции (не члены класса):**

- `findMatchingCloseTag()` — поиск закрывающего тега
- `extractAttribute()` — извлечение атрибута
- `removeFormattingTags()` — удаление тегов форматирования

## Методы и функции

### Публичные методы Table

#### void Table::addRow(const std::vector<Cell>& row)
**Описание:** Добавляет строку ячеек в таблицу.  
**Параметры:** `row` — вектор ячеек для добавления.  
**Сложность:** O(1) амортизированная (добавление в конец вектора).  
**Исключения:** Не генерирует исключений (при недостатке памяти может быть исключение от `std::vector`).

#### const std::vector<std::vector<Cell>>& Table::getRows() const
**Описание:** Возвращает константную ссылку на все строки таблицы.  
**Возвращаемое значение:** Константная ссылка на контейнер строк.  
**Использование:** Для чтения структуры таблицы без копирования данных.  
**Примечание:** Не позволяет модифицировать таблицу.

#### void Table::clear()
**Описание:** Очищает таблицу, удаляя все строки.  
**Действия:** Вызывает `rows.clear()`.  
**Использование:** При очистке данных в интерфейсе.

#### std::string Table::toHtml() const
**Описание:** Преобразует таблицу в HTML-код.  
**Возвращаемое значение:** Строка с валидным HTML-кодом таблицы.  
**Формат:** Стандартный HTML-код с тегами `<table>`, `<tr>`, `<td>`, `<th>`.  
**Особенности:**
- Добавляет атрибут `border="1"` для видимости границ
- Атрибуты `rowspan` и `colspan` добавляются только если > 1
- Использует `<th>` для заголовков (`isHeader == true`)
- Форматирует вывод с отступами для читаемости

### Статические методы TableParser

#### Table TableParser::parseSimpleHtml(const std::string& html)
**Описание:** Парсит HTML-код таблицы в объект `Table`.  
**Параметры:** `html` — HTML-код таблицы (ожидается уже очищенный).  
**Возвращаемое значение:** Объект `Table` с распарсенными данными.  
**Алгоритм:**
1. Поиск всех тегов `<tr>`
2. Для каждой строки поиск всех `<td>` и `<th>`
3. Извлечение атрибутов и содержимого ячеек
4. Очистка содержимого через `cleanCellContent()`
5. Создание объектов `Cell` и добавление в таблицу

**Отладочный вывод:** Информация о количестве строк и ячеек.  
**Исключения:** Может возникнуть при ошибках парсинга (обрабатывается в вызывающем коде).

#### std::string TableParser::extractTableFromWordHtml(const std::string& html)
**Описание:** Извлекает таблицу из полного HTML-кода Word-документа.  
**Параметры:** `html` — полный HTML-код документа.  
**Возвращаемое значение:** HTML-код только таблицы или пустая строка.  
**Алгоритм:**
1. Нормализация HTML
2. Поиск первого `<table>`
3. Поиск соответствующего `</table>`
4. Извлечение подстроки
5. Удаление Word-стилей

#### std::string TableParser::cleanWordHtml(const std::string& html)
**Описание:** Главный метод для комплексной очистки Word HTML.  
**Параметры:** `html` — полный HTML-код Word-документа.  
**Возвращаемое значение:** Чистый HTML-код таблицы.  
**Алгоритм:**
1. Извлечение таблицы
2. Парсинг в объект `Table`
3. Преобразование обратно в HTML
4. Возврат результата

**Использование:** Вызывается из интерфейса при нажатии кнопки "Очистить HTML".

#### std::string TableParser::cleanCellContent(const std::string& content)
**Описание:** Очищает содержимое ячейки от форматирования.  
**Параметры:** `content` — исходное содержимое ячейки.  
**Возвращаемое значение:** Очищенное содержимое.  
**Этапы:**
1. Удаление тегов форматирования
2. Обработка параграфов
3. Удаление HTML-сущностей
4. Нормализация пробелов
5. Обрезка пробелов по краям

**Сложность:** O(n), где n — длина входной строки.  
**Особенности:** Сохраняет структуру параграфов, но удаляет все форматирование.

## Работа приложения

### Поток выполнения

1. **Запуск приложения:**
   - `main()` создает `QApplication`
   - Создается и отображается `MainWindow`
   - Запускается цикл обработки событий Qt

2. **Загрузка HTML:**
   - Пользователь нажимает "Открыть" или вставляет код вручную
   - HTML-код загружается в `sourceTextEdit`

3. **Очистка HTML:**
   - Пользователь нажимает "Очистить HTML"
   - Вызывается `TableParser::cleanWordHtml()`
   - HTML проходит через этапы:
     - Извлечение таблицы
     - Нормализация
     - Удаление стилей
     - Парсинг в структуру `Table`
     - Очистка содержимого ячеек
     - Преобразование обратно в HTML
   - Результат отображается в `resultTextEdit`

4. **Сохранение/Копирование:**
   - Пользователь может сохранить результат в файл
   - Или скопировать в буфер обмена
   - Данные берутся из `resultTextEdit`

### Обработка данных

**Входные данные:**
- HTML-код с таблицей (может быть из Word, с множеством стилей и атрибутов)

**Обработка:**
- Извлечение структуры таблицы
- Удаление ненужных элементов
- Сохранение структуры (строки, ячейки, rowspan, colspan)
- Очистка содержимого ячеек

**Выходные данные:**
- Чистый HTML-код таблицы
- Минимальное форматирование
- Сохраненная структура

### Обработка ошибок

Все критические операции в `MainWindow` обрабатывают ошибки:
- **Файловые операции:** Проверка успешности открытия/записи
- **Парсинг:** Перехват исключений через `try-catch`
- **Пользовательский интерфейс:** Информативные сообщения об ошибках
- **Статусная строка:** Отображение текущего состояния операции

## Заключение

Проект **HTML Table Cleaner** представляет собой специализированное приложение для очистки HTML-таблиц с акцентом на работу с таблицами, экспортированными из Microsoft Word. Приложение использует современные технологии Qt и C++17, предоставляет удобный пользовательский интерфейс и эффективно обрабатывает сложный HTML-код, преобразуя его в чистую, читаемую структуру.

Код проекта хорошо структурирован, разделен на логические модули (парсинг, представление данных, интерфейс), что облегчает понимание и поддержку. Все основные функции документированы и обрабатывают ошибки, что обеспечивает стабильность работы приложения.
